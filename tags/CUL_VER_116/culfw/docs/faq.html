1. My CUL was initialized with X01 or X09 to receive data but there is no output at all. I am sure that there is traffic on air. CUL responds as expected to other commands like l and V. Is my CUL defect?
    Please check the response to command C35. Reception mode is turned on if the return value is 13. If reception mode is enabled but nothing is received, the EEPROM data is probably corrupt. You can reinitialize the EEPROM data with the default setting from program code by means of the command W01FFFF.
